Sobre Mim

Sou um desenvolvedor (me considero autodidata) com experiência em desenvolvimento de software, tanto em front-end quanto em back-end, além de ter conhecimentos em nuvem (AWS EC2). Utilizo tecnologias como TypeScript, Golang, Linux, MySQL, Git, AWS, também já utilizei Rabbitmq para um projeto pessoal, onde ele era responsável por salvar os dados de um cliente em uma API de pagamentos.
